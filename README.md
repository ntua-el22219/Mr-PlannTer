# Mr. PlannTer — README Παράδοσης

Το Mr. PlannTer είναι μια απλή εφαρμογή οργάνωσης εργασιών (tasks & deadlines) και ημερολογίου, υλοποιημένη σε Flutter.

— α) Σύντομες οδηγίες για εγκατάσταση και χρήση —

Προαπαιτούμενα (για απλό χρήστη, εγκατάσταση μέσω APK):
- Μία συσκευή Android με λειτουργικό Android 5.0+ (API 21 ή νεότερο).
- Σύνδεση στο διαδίκτυο προαιρετικά (μόνο για Google Sync).
- Ενεργοποίηση «Εγκατάστασης από άγνωστες πηγές» όταν ζητηθεί.

Τρόπος 1 — Εγκατάσταση μέσω APK (προτείνεται για μη ειδικούς):
1. Κατεβάστε το αρχείο APK από τον παρακάτω σύνδεσμο: [APK Link – να συμπληρωθεί].
2. Μεταφέρετε το APK στη συσκευή (ή κάντε drag-and-drop σε Emulator).
3. Πατήστε πάνω στο APK και ολοκληρώστε την εγκατάσταση (αποδεχθείτε τυχόν προτροπές για «άγνωστες πηγές»).
4. Ανοίξτε την εφαρμογή «Mr. PlannTer» από το εικονίδιο.

Τρόπος 2 — Εκτέλεση από κώδικα (για developers):
1. Εγκαταστήστε Flutter (3.22+), Android Studio και Android SDK Platform 36.
2. Κλωνοποιήστε το αποθετήριο:
	- `git clone <REPO_LINK>`
	- `cd Mr-PlannTer`
3. Εγκαταστήστε εξαρτήσεις: `flutter pub get`
4. Εκτέλεση σε συσκευή ή emulator: `flutter run`
5. Δημιουργία APK (release): `flutter build apk --release`
	- Το APK θα βρεθεί στο: `build/app/outputs/apk/release/app-release.apk`

Χρήση (βασικά βήματα):
- Πλοήγηση: Στην αρχή βλέπετε την «Today/Month/Calendar» προβολή. Μπορείτε να αλλάξετε προβολή με τα κουμπιά επάνω ή με οριζόντιο swipe.
- Προσθήκη εργασίας: Στην προβολή Calendar, πατήστε το σύμβολο «+» σε ώρα/slot για δημιουργία Task ή Deadline. Ορίστε τίτλο, ημερομηνία/ώρα και διάρκεια.
- Εμφάνιση εργασιών: Οι εργασίες εμφανίζονται χρωματικά ανάλογα με την «σημαντικότητα» τους.
- (Προαιρετικό) Google Calendar Sync: Στην προβολή Calendar, επιλέξτε «Google Calendar Sync» για Export/Import. Αν δεν θέλετε συγχρονισμό, μπορείτε να χρησιμοποιήσετε πλήρως την εφαρμογή χωρίς Google.

Σημειώσεις απλοποίησης:
- Δεν απαιτείται βάση δεδομένων/παραμετροποίηση server· όλα τα δεδομένα αποθηκεύονται τοπικά στη συσκευή.
- Δεν απαιτούνται ρυθμίσεις firewall ή σύνθετες βιβλιοθήκες από τον χρήστη.

— β) Android SDK / Image & Σύνδεσμοι —

Απαιτήσεις SDK (σύμφωνα με το project Android build):

Emulator image (προτεινόμενο):
- Android 15 (API 35) ή νεότερο, με Google APIs (κατά προτίμηση εικόνα με Play Store).

Troubleshooting (σύντομο):
- Αν η εγκατάσταση APK μπλοκάρεται, ενεργοποιήστε «Άγνωστες πηγές» και βεβαιωθείτε ότι υπάρχουν επαρκείς δικαιοδοσίες/χώρος.
- Για Google Sync απαιτείται Google Play Services και είσοδος σε Google λογαριασμό στη συσκευή. Αν η αυθεντικοποίηση αποτύχει, συνεχίστε χωρίς sync.
